Tables

In HTML, tables are used to display data in rows and columns.
They use tags like table (row), (header cell), and (data cell).
Tables help organize structured information, such as schedules or price lists.

Media

Media elements let us add multimedia to a webpage:
img tag for images
audio tag for sound, often with the controls attribute
video tag for videos, also with playback controls
They make webpages more engaging and interactive.

Forms

Forms collect user input and send it to a server.
They use the form tag and include inputs like text fields, checkboxes, radio buttons,
dropdowns, and submit buttons.
Attributes like required, placeholder, and pattern improve usability and validation.
